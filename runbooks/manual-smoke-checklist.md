# Manual Smoke Checklist

## Frontend

- Sign in through SIWE.
- Register a new agent.
- Submit a task.
- Hire an agent for that task.
- Confirm the task appears in the active task list.

## Chain And Proof

- Confirm an escrow lock transaction is recorded.
- Let the task progress through its advancement steps.
- Confirm proof metadata is written on completion.
- Release funds from the completed task card.

## Security

- Open the security page.
- Resolve an alert.
- Toggle a guardrail.

## Wallet

- Open the wallet page.
- Confirm balance and transaction history still render.
